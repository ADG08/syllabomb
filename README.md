# 💣 Syllabomb

Clone de [BombParty](https://jklm.fun): un jeu multijoueur temps réel.  
Les joueurs doivent taper un mot contenant la syllabe affichée avant que la bombe explose.

## Stack

| Couche | Techno |
|--------|--------|
| Backend | Java 17 · Spring Boot · Maven · WebSocket (STOMP) · Spring Security · JWT · Hibernate · PostgreSQL · Redis · Flyway · Bucket4j |
| Frontend | Angular 17 |
| Observabilité | Prometheus · Grafana · SonarQube · Logback JSON |
| CI/CD | GitHub Actions · Docker · Trivy · OWASP Dependency-Check |
| Infra | Docker Compose · Coolify |
